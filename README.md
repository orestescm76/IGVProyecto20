# IGVProyecto20
## Proyecto de IGV 2020.
Se trata de un programa que crea una escena con varios objetos y a esos objetos se les aplica un color y una textura.
* Objetos:
	* Cilindro (malla de triángulos)
	* Quad
	* Tetera (primitiva)
	* Caballo (OBJ Externo)
* Texturas:
	* Cacodemonio (DOOM)
	* Mirage
	* Windows 95
	* Piedra
	* Mármol
